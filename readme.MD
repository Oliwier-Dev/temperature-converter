# Temperature Converter

A small browser-based temperature converter built with vanilla JavaScript.
It converts values between Celsius, Fahrenheit, and Kelvin using form input.

## Features

- Convert between Celsius, Fahrenheit, and Kelvin
- Simple and responsive UI
- Runs directly in the browser (no build tools)

## How to run

Live demo: https://oliwier-dev.github.io/temperature-converter/

1. Enter a numeric value.
2. Choose the source unit and target unit.
3. Click `Submit` to see the converted result.

- `index.html` - app structure and styling
- `script.js` - conversion logic and form handling
